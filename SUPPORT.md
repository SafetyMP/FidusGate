# Support

FidusGate is an **open-source reference implementation** for AI agent governance. Support expectations:

| Channel | Use for |
|---------|---------|
| [GitHub Issues](https://github.com/SafetyMP/FidusGate/issues) | Bugs, feature proposals, documentation gaps (use templates) |
| [SECURITY.md](SECURITY.md) | Private vulnerability reports — never public issues |
| [CONTRIBUTING.md](CONTRIBUTING.md) | Local setup, tests, PR checklist |
| [docs/README.md](docs/README.md) | Architecture, walkthroughs, SME playbooks |

There is no paid SLA or production support for this reference repo. Best-effort maintainer review applies to pull requests that pass `npm run test` and follow the PR template.
